New Web Root Created

You may want to start by customizing the Apache configuration. You can use a2enmod to enable modules and a2ctl to restart Apache. Configure your site by modifying /web/quizmo/etc/apache2/sites-enabled/00-default. Then modify or delete this file, it is located at /web/quizmo/app/public/index.html

All of your application code should go in the app/ directory, with web-accessible files under app/public/. Things like libraries go in app/lib/, and all volatile data (temp files, pids, locks, cache, etc.) go in various places under var/.

If this is all new and scary, find enlightenment in the extensive Web Roots Documentation.
